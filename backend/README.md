# Setu AI Backend

FastAPI backend for Setu AI retail assistant.

## Quick Start

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run server:
```bash
uvicorn main:app --reload
```

3. API will be available at: http://localhost:8000
4. API docs at: http://localhost:8000/docs

## Data Files

- `data/products.json` - Product catalog
- `data/inventory.json` - Stock information
- `data/user_profiles.json` - User data

## Endpoints

See main.py for all available endpoints.
